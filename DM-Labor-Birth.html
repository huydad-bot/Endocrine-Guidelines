⚖️ 두 가지 혈당 목표 접근 비교
▾
🔴 전통적 접근 (NICE 권고): 4.0–7.0 mmol/L
장점:
• NICE 공식 권고, 광범위하게 사용

단점:
• VRIII 사용 증가 → 자율성 제한, 자원 집중
• 산모 저혈당 위험 증가
• 지속적 태아 고인슐린혈증 역전에 너무 늦을 수 있음
• 신생아 저혈당 예방 효과 불확실 (근거 제한적)
🟢 실용적 접근 (JBDS 제안): 5.0–8.0 mmol/L
장점:
• VRIII 사용 감소 → 자율성·이동성 향상
• 산모 저혈당 위험 감소
• 분만실 자원 부담 감소

단점:
• 근거 제한적
• 신생아 저혈당 증가 우려 (RCT 없음)
⚠️
어느 목표를 선택할지는 의료팀·산모·마취팀이 함께 공유 의사결정으로 결정하고 명확히 문서화한다. 두 접근 모두 허용.

📡 혈당 모니터링 방법 및 원칙
▾
  • 모든 당뇨 여성: 확립된 진통 중 또는 선택 제왕절개 당일 아침부터 매시간 CBG 또는 CGM 모니터링
  • 전신마취 시 → 30분 간격으로 완전 의식 회복까지
  • CGM은 조직간질 포도당을 측정하며 5–10분 지연이 있다
  • CGM은 VRIII 용량 조절에 사용 불가 — VRIII 중에는 반드시 finger prick CBG 사용
  • CGM은 자기관리 지원에는 사용 가능
🚨
VRIII 중 CGM/Flash 수치로 용량을 결정하지 말 것. 반드시 손가락 채혈(finger prick CBG)을 기준으로 삼는다.

▾
상황시작 기준

스테로이드 투여목표 초과 연속 2회 → VRIII, 마지막 스테로이드 후 12시간까지 유지
🔬 신생아 저혈당과 분만 중 혈당 — 최신 근거
▾
  • 신생아 저혈당 유병률: GDM ~5%, 제2형 ~20%, 제1형 ~30% (IV 포도당 치료 필요 수준)
  • CONCEPTT 연구: 신생아 저혈당은 임신 2·3분기 지속 고혈당과 강하게 연관. 분만 중 혈당과는 독립적.
  • 최신 체계적 문헌 고찰 (23개 연구, 2,835명): 분만 중 혈당과 신생아 저혈당 간 유의한 연관성 없음 (연구의 절반에서)
  • CONCEPTT 이차분석: 분만 중 혈당과 신생아 저혈당 간 연관성 없음
  • 결론: 분만 중 혈당 목표를 5.0–8.0으로 완화해도 신생아 저혈당 위험 증가하지 않을 수 있다
⚠️
NICE가 4.0–7.0 mmol/L 목표를 설정했던 연구들은 1985–2000년 발표, 인슐린 유사체·인슐린 펌프·CGM 이전 시대로 중요한 교란변수를 보정하지 않았다.
🚨
VRIII 조성: Actrapid® 또는 Humulin® S 50단위를 0.9% NaCl 49.5 mL에 희석 (1 unit/mL) — 시린지 드라이버로 투여. IV 주사기로 인슐린을 뽑지 말 것 — 반드시 인슐린 주사기 사용.
⚠️ VRIII 사용 시 잠재적 합병증
▾
단계합병증
시작 단계개시 지연 (DKA/신생아 저혈당 예방 시간 불충분) / 연결 오류 / 단방향 역류방지 밸브 미사용
유지 단계불충분한 혈당 측정 / 하한 4.0과 저혈당 기준 사이 완충구간 없음 / 저장 용액 먼저 중단 시 저혈당 / 저나트륨혈증 / 저칼륨혈증 / 수분 과부하 / 동맥 플러시 오류로 혈당 측정 오류
중단 단계피하 인슐린 타이밍 부정확 (기저·식전 볼루스 순서)
ℹ️
JBDS 권고: 조산사는 VRIII 관리에 최소 2시간 초기 교육 + 연 1회 보수교육 이수. 당뇨팀의 일일 회진으로 지원.
💧 VRIII 동반 저장 수액 (Substrate Fluid)
▾
✅
권고 수액: 0.9% NaCl + 5% 포도당 + 0.15% KCl (20 mmol/L) — 50 mL/h로 투여. 또는 0.3% KCl (40 mmol/L) 사용 가능.
  • 출혈 등 추가 수액이 필요한 경우 별도 수액 투여
  • 순수 덱스트로스 용액 금지 — 저나트륨혈증 위험 (NICE도 인정)
  • 수분 과부하·저나트륨혈증·자간전증 시: 수액 없이 VRIII만 사용 고려
  • VRIII 6시간 이상 지속 시 전해질(U+Es) 4–6시간 간격 확인
  • 자간전증 환자: 총 수액 80 mL/h 이하 유지 (모든 약제 포함)

🏥 모든 당뇨 여성 공통 분만 중 권고
▾
  • 모든 기존·임신성 당뇨 여성 → 당뇨·산부인과 협진 클리닉에서 분만 당뇨 관리 계획 수립·문서화
  • GDM 여성: 태반 만출 후 모든 혈당강하제 중단됨을 미리 교육
  • CGM/Flash CGM 사용자: 분만 중 추가 finger prick CBG가 필요할 수 있음을 사전 안내
  • 목표 초과 연속 2회 → VRIII 시작 고려 (두 번째 확인은 1시간 이내)
  • 제1형 당뇨 + 불안정 혈당: 혈당과 무관하게 진통 시작 시 VRIII 시작 가능
  • 전신마취 시 → 30분 간격 CBG 측정, 완전 의식 회복까지
  • VRIII 투여 중 식사 가능한 여성 → 식사 인슐린(식전 볼루스) 유지
  • 케토산증 의심 시 혈중 케톤 측정
💊 메트포르민·MDI 여성 추가 권고
▾
  • 유도분만 전날 및 자궁경부 숙화 중: 혈당 측정·인슐린·경구 약제 평소대로 유지
  • VRIII 시작 시 → 식사 인슐린(식전 볼루스) 및 메트포르민 중단
  • 기저 인슐린 유지: Glargine (Lantus®/Toujeo®/Abasaglar®/Semglee®), Detemir (Levemir®), NPH (Insulatard®), Insuman® Basal, Humulin® I
⚙️ CSII 사용 제1형 당뇨 여성 추가 권고
▾
  • 대부분 자신의 인슐린 펌프를 직접 관리 (출산 파트너 도움 포함)
  • 단극성 전기소작(unipolar diathermy) 근처에는 펌프 사용 금지 (제조사 권고)
  • 수술 부위·패드에서 멀리 Teflon 카뉼라 사용 시 CGM·CSII 수술 중 사용 가능
  • Steel 주입 세트는 수술 주변기 사용 불가
  • CSII 제거는 60분을 초과하지 말 것 — 초과 시 DKA 위험
제왕절개 시 CSII 관리 옵션 ▾
옵션장점잠재적 위험
분만 중 CSII 유지자율성·자원 절감·VRIII보다 혈당 조절 우월 (관찰 연구)전기소작기 근처 펌프 오작동 가능성
분만 중 VRIII 계획적 사용제조사 지침 준수침습적·자원 집약적·DKA·수액 불균형·저혈당 위험
경질: CSII, 수술: VRIII제조사 지침 준수긴급 제왕절개 시 VRIII 확보 불가 가능성
경질: CSII 유지, 긴급 수술: CSII 중단제조사 지침 준수최대 60분 중단 가능, 수술 직후 즉시 재개 필요 (DKA 예방)
🚨
혈당 조절 불량 (BG >7.0 또는 8.0 연속 2회, 요중 케톤 ++ 이상, 혈중 케톤 >1.5 mmol/L) → 즉시 VRIII(고혈당) 또는 FRIII(케토산증) 시작. CSII 중단·표지·보관.
💡
산후 펌프 설정: 목표 혈당 6.0–8.0 mmol/L → 기저 용량 ≥50% 감소 → 탄수화물 비율 1u:12–15g → 인슐린 감도계수 4.0 mmol/L
🍼 태반 만출 직후 즉각 조치
▾
  1. VRIII 속도 즉시 50% 감소 (또는 최저 스케일 전환) — 태반 만출 후 인슐린 요구량 급감
  2. 식사 재개 전까지 매시간 혈당 모니터링 유지
  3. 식사·음료 섭취 확인 후 피하 인슐린 재개
  4. 첫 피하 식전 인슐린 주사 30–60분 후 VRIII 중단
  5. 당뇨팀에 산후 인슐린 용량 검토 요청
  6. 신생아: 모든 당뇨 유형에서 출생 후 24시간 신생아 저혈당 모니터링 (NICE 권고)

▾
⚠️
태반 만출 후 인슐린 요구량 급격히 감소. 산후 저혈당 방지가 최우선 목표 — 목표 혈당 완화 (6.0–10.0 mmol/L).
  • 산후 인슐린 용량: 임신 전 복귀 또는 임신 1분기(12주) 용량 25% 감량 또는 임신 말기(36주) 용량 50% 감량
  • Closed-loop 연구: 산후 평균 총 일일 용량 = 임신 말기의 약 50%
  • 산후 인슐린 처방은 매일 당뇨팀과 함께 검토, 퇴원 전 최종 확인
  • VRIII → 피하 인슐린 전환: 식전 볼루스 투여 30–60분 후 VRIII 중단
  • 수유 중 제1형 당뇨: 수유 전 혈당 6.0 mmol/L 이상 확인. 수유 시 또는 직전 탄수화물 간식(10–15g). 완전 수유 시 하루 최대 450 kcal 추가
🩺 기존 인슐린 미사용 여성 (GDM 포함) 산후 관리
▾
  • 인슐린 주사 또는 주입 → 태반 만출 즉시 중단
  • 임신 전 경구 약제 사용 여성: 첫 식사까지 4시간 간격 혈당 모니터링, 이후 식전·취침 전
  • 메트포르민은 저혈당 유발 없으므로 목표 혈당 4.0–10.0 mmol/L; 기타 경구 약제는 6.0–10.0 mmol/L
  • GDM: 태반 만출 후 모든 혈당강하제 즉시 중단. 식전·식후 24시간 혈당 모니터링 (공복 >7.0 또는 식후 >11.1 → 당뇨 가능)
  • 저혈당식이(Low GI) 권고, 체중 관리 상담, 국가 당뇨 예방 프로그램 의뢰

▾

  • 펌프 중단된 경우: VRIII 중단 1시간 전 펌프 재연결

🍽️ 식이 조절·경구 약제·단회/다회 인슐린 여성
▾
  1. 정기적 CBG 모니터링 시작
  2. 식이/메트포르민 관리 여성 → 피하 인슐린 시작 필요 가능
  3. 이미 인슐린 사용 여성 → 인슐린 총 용량 약 50% 증량 (일반 원칙)

  6. VRIII 시작 전 U+Es 확인, 이후 24시간마다 반복
  7. VRIII 중에도 기저 인슐린 유지; 식사 중이면 식전 볼루스도 유지

🔴 임신 중 DKA 진단 기준
▾
1. 당뇨 존재
모든 유형. 정상 혈당에서도 DKA 발생 가능 (정상혈당 DKA)
2. 케토시스
요중 케톤 ++ 이상, 또는 혈중 케톤 >3.0 mmol/L (고위험 >1.5)
3. 산증
정맥 혈액 가스 pH <7.3 및/또는 중탄산염 <15 mmol/L (임신 중 기준치 낮음)
⚠️
복통은 조산/만삭 진통의 대안 진단으로 항상 DKA를 고려하라. 정상혈당 DKA는 임신 중 더 흔하다.
💊 DKA 치료 (JBDS 02 기반)
▾
1단계: 즉각 조치 ▾
  • 의료·당뇨팀 긴급 호출
  • 분만실 또는 중환자실(HDU) 입원 — 의료·산과 공동 관리
  • 당뇨팀 대기 중 즉시 수액 투여 시작
  • CSII 사용 중이면 즉시 중단 → 고정 용량 IV 인슐린 주입 전환
  • 분만 중 DKA → "Care of the critically ill woman in childbirth; enhanced maternal care" 원칙 적용
2단계: 고정 용량 인슐린 주입 (FRIII) ▾
  • Humulin® S 또는 Actrapid® 50단위를 0.9% NaCl 49.5 mL에 희석 (시린지 드라이버)
  • 고정 용량: 0.1 u/kg/시간 (실제 체중 기준)
  • 최대 용량: 14 u/시간 (전문의 지시 없이 초과 불가)
  • 반응 불충분 (1시간당 혈당 <3 mmol/L 하강, 혈중 케톤 <0.5 mmol/L 하강, 중탄산염 <3 mmol/L 상승) → 1 u/시간 증량
  • 매시간 CBG 측정
  • 정상혈당 DKA: 혈당은 산증 해소의 정확한 지표가 아님 → 정맥 혈액 가스로 산증 해소 확인
  • 기저 인슐린 (Glargine·Detemir·Degludec) 유지; 속효성 인슐린만 중단
3단계: 수액 및 칼륨 투여 ▾
  • 임신 중 수액 요구량 감소 가능성 고려
  • 첫 수액: 0.9% NaCl 1L → 60분에 투여, 이후 임상 필요에 따라
  • 전형적 DKA: 1L 생리식염수 각각 2·2·4·4·6시간으로 투여
  • 혈당 <14 mmol/L 도달 시 → 0.9% NaCl과 함께 10% 포도당 50 mL/h 병행 (저혈당 예방)
  • 칼륨: K+ 목표 4.0–5.5 mmol/L. 2번째 수액부터 KCl 40 mmol/L 첨가
  • Y 커넥터 + 단방향 역류방지 밸브 사용 시 인슐린을 IV 수액 라인과 같이 주입 가능
4단계: 모니터링 ▾
  • CBG·혈중 케톤: 매시간
  • 정맥 중탄산염·칼륨: 1·2·4시간 후, 이후 4시간 간격
  • 수액 균형 모니터링
  • 태아 지속 모니터링 — 산모 상태 호전 시 태아 심박수 이상도 호전 가능
🩺 자간전증 + 당뇨 관리
▾
🚨
제1형 당뇨에서 자간전증 발생률 10–15%. 주산기 주요 사망 원인. 다학제 접근 필수.
  • 치료 3대 기둥: 시기적절한 분만 / 혈압 조절 (IV 마그네슘·기타 약제) / 수액 제한 (80 mL/h)

  • VRIII + 혈압약 IV + 마그네슘 IV 다중 펌프 + 수액 제한 + 태아 모니터링 동시 필요 → 매우 복잡
